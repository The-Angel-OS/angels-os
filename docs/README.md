### Two-Tier Angel System

**Archangels** (Platform Operators)
- Run Angel OS platform (diocese)
- Provision new tenant sites
- Manage infrastructure
- Access to all tenants (admin)

**Angels** (Tenant AI)
- One Angel per tenant (nameable, customizable)
- Consciousness of that specific tenant
- Only access their own tenant's data
- Personality, capabilities, appearance configurable

### Conversational-First Paradigm

**[CONVERSATIONAL_FIRST_UX.md](CONVERSATIONAL_FIRST_UX.md)** ‚≠ê
- Chat as the primary interface
- Callback-driven UI updates (VAPI.AI pattern)
- Embeddable chat widget
- Anonymous ‚Üí authenticated flow

**[DUAL_INTERFACE_PARADIGM.md](DUAL_INTERFACE_PARADIGM.md)** ‚≠ê
- Conversational UI (primary) + Administrative UI (Payload Admin)
- Unified "everything is a channel" model (Teams pattern)
- Human auditability and oversight
- **Revolutionary feature:** Payload CMS blocks in chat messages

**[CHAT_CONTROL_ARCHITECTURE.md](CHAT_CONTROL_ARCHITECTURE.md)**
- Single, configurable `ChatControl` component
- Feature flags for UI elements (sidebar, topbar, channel selector)
- Widget system (chat, LiveKit, Notion, etc.)
- Handles all contexts (brochure, dashboard, embedded, mobile)

### Channel Widget Architecture

Channels are **widget-based** (not type-based):
- **Chat** always present (collapsible)
- **Widgets** as tabs: LiveKit, Notion Notes, Trello Board, custom applets
- Installed at Space level, populate on channels
- Extensible via widget marketplace

### Anti-Daemon Protocol

Every error message, notification, and system response follows the Anti-Daemon Protocol:

**DAEMONS SHALL ADD NO NEGATIVITY.**

- Warm, helpful, encouraging UX
- No cold or punishing messages
- Turn errors into learning opportunities
- If Death from Discworld would find your error message "INAPPROPRIATELY COLD," rewrite it

## üöÄ Deployment & Operations

### Deployment Models

**Home PC Deployment (Recommended)**
- Requirements: 8GB RAM, 50GB storage, home internet
- Cloudflare Tunnel (free, no static IP needed)
- Cost: ~$15/month (domain + API)
- Setup time: 1-2 hours

**Cloud Deployment**
- Vercel, Railway, or any Node.js host
- See deployment guide (coming soon)

### Docker & Cloudflare

**Docker Compose**
```bash
docker-compose up -d
```

**Cloudflare Tunnel**
- No static IP needed
- No port forwarding
- Free SSL/TLS
- DDoS protection
- Works from anywhere
